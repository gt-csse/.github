# Georgia Tech's Center for Scientific Software Engineering (CSSE)

This Github repository is the central location for public resources developed by [Georgia Tech's Center for SSE](https://ssecenter.cc.gatech.edu/). Please check our pinned repositories for more details on best practices and resources for software engineering of scientific applications.

The Center for Scientific Software Engineering (CSSE) at Georgia Tech is an effort that is part of the [Virtual Institute for Scientific Software (VISS)](https://www.schmidtsciences.org/viss/). VISS was launched by Schmidt Sciences to address the need for training and educating software engineers, and the CSSE at Georgia Tech is focused on developing new methodologies for improving high-performance scientific codes and for training tomorrow’s software engineers.
